# [25-W SQL 스터디] 6주차 공부

## 1. [서울에 위치한 식당 목록 출력하기](https://school.programmers.co.kr/learn/courses/30/lessons/131118)
공부한 날짜: 2025년 2월 12일

### 작성한 SQL 쿼리
```SQL
SELECT
    a.REST_ID,
    a.REST_NAME,
    a.FOOD_TYPE,
    a.FAVORITES,
    a.ADDRESS,
    ROUND(AVG(b.REVIEW_SCORE), 2) AS SCORE
FROM REST_INFO a
JOIN REST_REVIEW b ON a.REST_ID = b.REST_ID
GROUP BY a.REST_ID
HAVING a.ADDRESS LIKE '서울%'
ORDER BY
    SCORE DESC,
    a.FAVORITES DESC;
```
